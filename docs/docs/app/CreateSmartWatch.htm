CreateSmartWatch

Hello World

Returns a SmartWatch object.

smw = app.CreateSmartWatch( type ) → app object: SmartWatch


The following methods are available on the SmartWatch object:

AddLayout( layout )
CreateImage( image, width, height, options, pxwidth, pxheight ) → SmartWatch object: Image
CreateLayout( type, options ) → SmartWatch object: Layout
CreateSensor( name, options ) → SmartWatch object: Sensor
CreateText( text, width, height, options ) → SmartWatch object: Text
GetVersion() → string
RemoveLayout( lay )
SetMenu( list, iconPath )
SetOnBack( callback )
SetOnHide( callback )

StopApp()
